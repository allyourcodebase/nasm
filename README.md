# nasm

This is [nasm](https://www.nasm.us/), packaged for [Zig](https://ziglang.org/).
Both the nasm and ndisasm binaries are provided.

> Note: A previous version of this repository forked nasm and replaced its build
> system with Zig. However, we now use an external nasm source with a clean
> `master` branch. The old `master` is still available in the `legacy` branch.
> See [#7](https://github.com/allyourcodebase/nasm/issues/7) for more
> information.
